# Pranayama Effects Directory

## Purpose
Create a structured directory of breath practices and their associated effects so the information can later support an app or bot experience.

## Fields To Track
For each practice, track:
- energetic level
- carbon dioxide
- nitric oxide
- nervous system
- guna
- nadi
- vayu

## Practices

### Unequal Ratio Breathing (Vishama Vritti)
- Energetic level: Significantly calming
- Carbon dioxide: Significant increase
- Nitric oxide: Mild increase
- Nervous system: Parasympathetic / dorsal vagal complex
- Guna: Tamas
- Nadi: Ida
- Vayu: Samana
