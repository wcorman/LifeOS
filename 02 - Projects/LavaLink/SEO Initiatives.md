# SEO Initiatives

## Goal
Grow awareness of LavaLink as a solution for accessing Obsidian vaults from the browser.

## Core Initiatives

### 1. Keyword Targeting
Target search terms around browser-based access to Obsidian and remote vault workflows.

Potential keywords:
- obsidian in browser
- access obsidian vault online
- obsidian web access
- open obsidian notes in browser
- self-hosted obsidian web access
- remote access obsidian vault
- obsidian vault from browser
- browser-based second brain

### 2. Website Content
Create clear, useful content that explains the problem LavaLink solves.

Content ideas:
- Landing page focused on browser access for Obsidian
- Feature page for remote vault access
- Comparison page: LavaLink vs other ways to access Obsidian remotely
- Blog post: How to access your Obsidian vault from a browser
- Blog post: Best ways to view Markdown knowledge bases online
- FAQ covering privacy, sync, self-hosting, and setup

### 3. Reddit Responses
Engage naturally in relevant Reddit threads where people ask about accessing Obsidian from the web, remote note access, or self-hosted knowledge tools.

Focus areas:
- Answer the actual question first
- Mention LavaLink only when it is genuinely relevant
- Keep responses helpful and non-spammy
- Track responses in [[02 - Projects/LavaLink/Reddit Responses]]

### 4. Supporting SEO Work
- Build internal links between product and educational pages
- Capture recurring user questions from Reddit and turn them into content
- Watch which keyword themes lead to the best engagement
- Reuse strong wording from community conversations in landing pages and FAQs

## Next Actions
- Finalize target keyword clusters
- Draft first browser-access landing page
- Start logging Reddit posts and comments in [[02 - Projects/LavaLink/Reddit Responses]]
