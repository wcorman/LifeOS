# SEO Roadmap

## Objective
Rank LavaLink for high-intent searches related to accessing Obsidian vaults from a browser.

## Existing Content Base
Already published:
- How to Access Obsidian Vault on the Web
- LavaLink vs Obsidian Publish

This means the next SEO work should expand coverage instead of duplicating what already exists.

## Phase 1: Capture the Core Intent
Focus on the most direct and highest-intent keyword cluster first.

### Priority Keywords
- obsidian in browser
- obsidian web access
- access obsidian vault online
- obsidian web version

### Deliverables
- Main landing page targeting browser-based Obsidian access
- FAQ section addressing work computer / no install / remote access use cases
- Strong internal links between core product pages and published blog posts

## Phase 2: Capture Problem-Aware Search Traffic
Build educational content around the real questions people ask.

### Priority Topics
- how to use obsidian on a work computer
- can i use obsidian through a browser
- how to view obsidian notes on the web
- how to access obsidian remotely
- access obsidian without installing the app

### Deliverables
- 2-4 blog posts
- one FAQ / help page
- one setup / explainer page

## Phase 3: Own the Trust + Technical Angle
Target the more technical audience that cares about ownership, privacy, and self-hosting.

### Priority Topics
- self-hosted obsidian web access
- private browser access to obsidian vaults
- online access to markdown knowledge bases

### Deliverables
- self-hosting page
- privacy / security explainer
- architecture / how-it-works page

## Content Order
1. Landing page
2. FAQ / use-case page
3. “How to use Obsidian on a work computer” article
4. “Can you use Obsidian through a browser?” article
5. “Best ways to access Obsidian remotely” article
6. Self-hosting / privacy pages

## Success Signals
- Rankings for direct browser-access keywords
- Search impressions for problem-aware queries
- More Reddit/community threads matching LavaLink positioning
- Better on-page engagement for landing and help content

## Notes
This should be approached like a niche intent-capture strategy, not a mass-volume SEO play. The goal is to dominate a narrow, highly relevant problem space.
