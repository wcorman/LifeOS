# Eagle Heights Estates - Visual Direction / Design System Brief

## Overall Direction
The visual direction for Eagle Heights Estates should feel:
- refined
- nature-connected
- spacious
- calm
- credible
- quietly aspirational

It should not feel:
- overly luxury
- flashy
- generic real estate
- corporate
- overly rustic

The goal is to create a brand and site experience that feels grounded and elevated at the same time.

## Brand Feeling
Eagle Heights Estates should communicate:
- room to breathe
- a quieter pace
- land and openness
- confidence without hype
- a modern but natural sense of quality

## Visual Keywords
- open
- spacious
- restrained
- modern
- organic
- soft contrast
- premium but understated

## Layout Principles
- Use generous whitespace
- Avoid cramped sections
- Let imagery carry emotional weight
- Keep content blocks easy to scan
- Use clear hierarchy and rhythm
- Favor strong section spacing over heavy decorative treatment

## Typography Direction
The typography should feel:
- elegant but readable
- modern with warmth
- confident without being aggressive

Suggested approach:
- A refined serif or lightly expressive display face for headlines
- A clean sans serif for body copy and UI

Headline qualities:
- spacious
- calm
- slightly elevated

Body copy qualities:
- clean
- highly readable
- neutral and supportive

## Color Direction
The palette should feel natural, grounded, and clean.

Suggested color families:
- warm off-whites / soft stone
- charcoal / deep gray for contrast
- muted earth tones
- subtle greens or sage-inspired accents
- optional warm beige / sand tones

Avoid:
- harsh black and white only
- overly saturated greens or blues
- flashy gold / faux-luxury palettes

## Imagery Direction
Imagery should emphasize:
- land
- openness
- horizon / sky
- quiet natural beauty
- spacious living potential
- site progress and credibility

Ideal image qualities:
- natural light
- wide framing
- calm compositions
- minimal clutter
- real rather than overly polished stock feeling

## UI / Component Direction
### Buttons
- simple, confident, not overly rounded or playful
- strong contrast but understated
- primary buttons should feel intentional and calm

### Cards
- clean and structured
- subtle borders or soft separation
- avoid heavy shadows or overly trendy glassmorphism

### Forms
- simple and low-friction
- spacious input fields
- clear labels
- professional, calm feel

### FAQ / Accordion
- minimal and easy to scan
- smooth, clear interaction
- avoid visual clutter

## Homepage Direction
The homepage should rely heavily on:
- strong hero image / visual atmosphere
- concise copy
- clean section rhythm
- premium spacing
- value pillars that feel editorial rather than salesy

## Listings Page Direction
Because the site will use a Mapplic lot map:
- the page should feel informative and polished
- the map should be treated as a feature, not an afterthought
- listings cards and map UI should feel visually consistent
- status styling for lots should be clear and elegant

## Contact Page Direction
The inquiry experience should feel:
- approachable
- calm
- trustworthy
- friction-light

It should encourage action without pressure.

## Design System Building Blocks
Recommended system elements to define early:
- heading scale
- body text scale
- spacing scale
- container widths
- button styles
- form styles
- card styles
- section padding rules
- image treatment rules
- color tokens

## Notes
- The strongest design differentiator may simply be restraint and clarity.
- This project should feel more like a refined land/lifestyle brand than a typical real estate sales page.
- The design should reinforce the emotional promise: more space, more privacy, and a quieter life near Saskatoon.
